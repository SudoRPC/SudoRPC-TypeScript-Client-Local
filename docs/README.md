Local client for SudoRPC Client

## Install

```sh
yarn add @sudorpc/client-local
yarn add @sudorpc/core # Peer Dependency
# Or
npm install @sudorpc/client-local --save
npm install @sudorpc/core --save # Peer Dependency
```
